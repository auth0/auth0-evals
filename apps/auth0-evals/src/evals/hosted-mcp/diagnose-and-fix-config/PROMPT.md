## Task

A developer reports that the **Notifications Worker** application cannot get tokens for the **Notifications Service** API. They think there might be a missing grant.

Please:
1. Find the **Notifications Worker** application
2. Find the **Notifications Service** resource server
3. Check whether a grant exists from the Notifications Worker to the Notifications Service
4. If the grant is missing, create it with the `send:notifications` scope

Tell me what you found and what action you took.
