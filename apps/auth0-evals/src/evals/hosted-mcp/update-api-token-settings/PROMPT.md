## Task

The **Inventory Service** API needs its token expiration settings updated for security compliance.

Please update the **Inventory Service** resource server to:
- Set the token lifetime to **1800 seconds** (30 minutes)
- Enable offline access (allow refresh tokens)

Confirm back with the resource server ID and the updated token lifetime.
